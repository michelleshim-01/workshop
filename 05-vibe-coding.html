바이브코딩 5단계 사이클

코드를 몰라도 됩니다. 기획하고, 방향을 잡고, Claude에게 맡기고, 확인하고, 반복하는 것 — 이게 바이브코딩입니다.

⏱ 40분

바이브코딩의 핵심 능력

코딩 지식은 필요 없습니다. 이 4가지 능력만 있으면 됩니다:

능력의미
① 자동화 대상 정의어떤 반복 업무를 자동화할지 구체적으로 파악하기
② 단계별 프로세스 설명입력 → 처리 → 출력 흐름을 AI에게 명확히 설명하기
③ 반복 개선 마인드셋한 번에 완벽하지 않아도 됨 — 5~10번 수정이 정상
④ 결과 검증AI 결과물을 내 업무 지식으로 맞는지 확인하기

5단계 사이클 개요

② 플랜

각 단계 상세

1
Planning

기획

GitHub 레포지토리를 먼저 만들고, Claude에게 기획 문서 초안을 작성해달라고 요청합니다. 초안을 직접 읽고 수정하며 디테일을 완성하세요.

⚠️ 기획이 가장 중요합니다
3일짜리 프로젝트라면 1~2일은 기획에만 투자하세요. 코드는 AI가 짜지만, 뭘 만들지는 내가 결정해야 합니다. 기획이 탄탄해야 AI에게 제대로 일을 시킬 수 있습니다.
예시 프롬프트
내가 만들고 싶은 것: 구글 폼으로 받은 비품 요청을
자동으로 취합해서 부서별 발주서를 만들어주는 도구야.
이걸 위한 기획 문서를 작성해줘.
2
Plan Mode

플랜

기획 문서를 바탕으로 페이즈별 작업 계획 MD 파일을 만들고, Plan Mode에서 방향이 맞는지 확인합니다.

예시 프롬프트
기획.md를 읽고 구체적인 작업 계획을
페이즈별로 나눠서 plan.md 파일로 만들어줘

계획이 내가 원하는 방향과 다르면 수정을 요청하세요. 모르는 기술 용어는 Claude나 GPT에게 쉽게 설명해달라고 하면 됩니다.

3
Action & Save

액션 + 저장

작업 계획의 1단계를 실행하고, 완료 후 반드시 저장합니다.

핵심 리듬:
① 코드 작성 ← Claude가 실행
② git commit push ← "저장해줘" 한 마디로
③ /clear ← 컨텍스트 초기화
④ /init ← CLAUDE.md 업데이트
⑤ git commit push ← 다시 저장
↳ 다음 작업 단위 시작
🎮 세이브 포인트
큰 작업 전에 항상 "현재 상태 저장해줘"를 먼저 실행하세요. 문제가 생기면 언제든 이 지점으로 돌아올 수 있습니다.
4
Verification

확인

Claude에게 결과물을 직접 확인시킵니다. MCP(Playwright/Chrome)가 연결되어 있으면 Claude가 직접 브라우저를 열어 테스트합니다.

예시 프롬프트
만든 페이지를 열어서 비품 요청 버튼이 잘 작동하는지 확인해줘
에러가 있으면 뭔지 알려줘
⚡ 병렬 작업 팁
Claude가 테스트하는 동안 여러분은 다음 단계의 플랜을 세울 수 있습니다. Claude가 끝나면 결과를 받고 다음 작업으로 넘어가세요.
5
Iterate

수정 + 반복

문제가 발견되면 작은 플랜을 세우고 수정합니다. 그리고 다시 저장→확인→수정 사이클을 반복합니다.

🔁 3번 규칙
같은 문제를 3번 이상 수정해도 해결되지 않으면, 과감하게 버리고 3단계(저장 지점)으로 돌아가서 다시 시작하세요. 트러블슈팅 내용을 MD 파일로 기록해두면 다음 시도에서 더 잘 됩니다.

자동화 후보 찾기

어떤 업무를 자동화하면 좋을까요? 아래 체크리스트에 3개 이상 해당하면 좋은 후보입니다.

  • 매주/매월 반복하는 작업
  • 복사-붙여넣기를 반복하는 작업
  • 같은 템플릿 문서를 계속 만드는 작업
  • 여러 곳에서 데이터를 모아 합치는 작업
  • 웹에서 정보를 수집하는 작업

Hook — 작업 완료 알림

여러 작업을 동시에 돌릴 때, 작업이 끝났는지 계속 화면을 보고 있을 필요가 없습니다.

Mac에서 작업 완료 알림 설정
"Claude야, 작업이 끝나면 Mac에서 소리로 알려줘.
say 명령어로 '클로드 작업 완료'라고 말해줘"

Hook을 설정하면 작업이 끝날 때마다 알림이 옵니다. 여러 Claude 창을 열고 각각 다른 작업을 시켜도 완료 시 알림을 받을 수 있어요.

MCP로 외부 서비스 연결

MCP(Model Context Protocol)를 설치하면 Claude가 외부 서비스를 직접 사용할 수 있습니다.

🔍 Playwright / Chrome — 브라우저 자동화
📗 Google Sheets — 스프레드시트 읽기/쓰기
💬 Slack — 메시지 자동 전송
📓 Notion — 문서 자동 생성
📅 Google Calendar — 일정 읽기/쓰기
🐙 GitHub — PR 생성, 코드 리뷰
💡 Claude Core Work
Claude Code보다 진입장벽이 낮은 Claude Core Work도 있습니다. Mac 데스크탑 앱에서 "다운로드 폴더 정리해줘"처럼 파일 관리 작업을 시킬 수 있습니다. 관심 있으면 Claude 앱에서 찾아보세요!
