Part 1

설치 & Claude Code란?

Claude Code가 무엇인지 이해하고, 내 컴퓨터에 설치해서 첫 대화를 나눠봅니다.

⏱ 40분

Claude Code란?

Claude Code는 터미널에서 동작하는 AI 에이전트입니다. 파일을 읽고, 코드를 작성하고, 명령어를 실행하며, 스스로 결과를 확인하고 수정합니다.

쉽게 말하면: "24시간 일하는 스마트 인턴"

🔄 에이전트 루프
Claude Code는 세 단계를 반복합니다:
① 컨텍스트 수집 → 프로젝트 구조와 파일을 읽고 이해
② 작업 실행 → 파일 생성/수정, 명령어 실행
③ 결과 검증 → 테스트 실행, 오류 확인, 필요시 반복

다른 AI 도구와 비교

도구 역할 자율성
GitHub Copilot 코드 자동 완성 (한 줄씩) 낮음
Cursor / Windsurf 편집기 안에서 AI 채팅 중간
Claude Code 여러 파일에 걸쳐 자율적으로 작업 높음

전통 개발 vs 바이브코딩

🏗️ 전통적인 개발

  • 설계 → 기초부터 차근차근
  • 완성도 높은 코드 지향
  • 수정 시 투입 인력·시간 큼
  • 전문 지식 필요

🌊 바이브코딩 (Vibe Coding)

  • 대화로 코드 생성 (비용 ≈ 0)
  • 러프하게 빠르게 만들고 폐기
  • 수정 비용이 거의 없음
  • 기획력이 핵심 역량
⚠️ 기술 부채 주의
바이브코딩은 빠르지만, AI 코드의 품질 편차가 큽니다. 장기 운영 시 기술 부채가 쌓이면 — 수정 지점을 알 수 없고, 보안 문제가 생기고, 버전업이 어려워집니다. Git으로 버전을 잘 관리하고, CLAUDE.md로 규칙을 잡아두는 것이 해결책입니다.

설치하기

아래에서 내 운영체제를 선택하세요.

1

Node.js 설치 확인

터미널을 열고 (Cmd+Space → "터미널") 아래를 입력하세요.

Terminal
node --version

v18.0.0 이상이 나오면 OK. 없으면 nodejs.org에서 LTS 버전을 다운로드하세요.

2

Claude Code 설치

Terminal
npm install -g @anthropic-ai/claude-code
3

PATH 설정 (명령어를 못 찾는다고 나오면)

Terminal
echo 'export PATH="$HOME/.claude/bin:$PATH"' >> ~/.zshrc
source ~/.zshrc
4

설치 확인

Terminal
claude --version

버전 번호가 나오면 설치 완료!

⚠️ PowerShell 필수
Windows에서는 반드시 PowerShell을 사용해야 합니다. CMD(명령 프롬프트)에서는 작동하지 않습니다. VS Code 터미널이 기본 PowerShell이면 바로 사용 가능합니다.
1

Node.js 설치 확인

PowerShell을 열고 아래를 입력하세요.

PowerShell
node --version

v18 이상이 없으면 nodejs.org에서 LTS 버전을 다운로드하세요.

2

Claude Code 설치

PowerShell
npm install -g @anthropic-ai/claude-code
3

PATH 수동 추가 (명령어를 못 찾으면)

시스템 환경변수에 아래 경로를 추가하세요:

추가할 PATH 경로
%USERPROFILE%\.claude\bin

시작 → 환경 변수 검색 → 사용자 변수 Path 편집 → 새로 만들기 → 위 경로 입력 → 확인 → PowerShell 재시작

4

설치 확인

PowerShell
claude --version

API 키 설정 & 비용 한도

1

Anthropic 콘솔에서 API 키 발급

console.anthropic.com → API Keys → Create Key

2

⚠️ 비용 한도 설정 (꼭 하세요!)

Settings → Billing → Monthly Spend Limit → $5~10 설정. 처음엔 적게 설정해두는 게 안전합니다.

3

또는 Claude Pro 구독 사용

claude.ai/pro 구독($20/월)이 있으면 API 키 없이 claude 실행 후 브라우저 로그인으로 사용 가능. Claude Max($100/월)는 사용량 제한이 거의 없습니다.

VS Code 연동

터미널만으로도 사용 가능하지만, VS Code에서 작업하면 파일 변경사항과 Git 내역을 한눈에 볼 수 있어 훨씬 편합니다.

1

Claude VS Code 확장 설치

VS Code → Extensions (Ctrl+Shift+X) → "Claude" 검색 → Anthropic 공식 확장 설치

2

첫 실행

VS Code에서 작업 폴더 열기 → 하단 터미널 열기 (Ctrl+`) → claude 입력

3

유용한 VS Code 설정

File → Auto Save 켜기 · View → Light Modern 테마 · 한국어 팩 설치 (선택)

오류 메시지해결 방법

VS Code 확장 작동 안 함 VS Code 완전 재시작 (Cmd/Ctrl+Shift+P → Reload Window)
