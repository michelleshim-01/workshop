사전 준비물 체크리스트

워크샵 당일 전에 아래를 미리 준비해 오시면 첫 40분을 훨씬 수월하게 지나갈 수 있어요.

  • 노트북 (Mac 또는 Windows) — 태블릿은 불가
  • GitHub 계정 — github.com 가입 (무료)
  • Anthropic 계정 — console.anthropic.com 가입 (무료)
  • API 키 발급 + 월 비용 한도 $5 설정
  • VS Code 설치 — code.visualstudio.com
  • Node.js 설치 (v18 이상) — nodejs.org
